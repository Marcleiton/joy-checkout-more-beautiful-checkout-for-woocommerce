== Description ==

This plugin completely changes the look of your store's Checkout, bringing a more beautiful and professional look, with Payment Steps on one page.


We use some files from the base [clicking here](https://github.com/CMB2/)

Find the premium version with more themes and features, [clicking here](http://joycheckout.com/)


= Compatibility =

It is mandatory to have Woocommerce installed [clicking here](https://br.wordpress.org/plugins/woocommerce/)

This plugin is also compatible with [WooCommerce Extra Checkout Fields for Brazil](http://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil/).
This plugin is also compatible with [Checkout Field Editor (Checkout Manager) for WooCommerce](https://br.wordpress.org/plugins/woo-checkout-field-editor-pro/).
This plugin is compatible with several payment methods, we have not tested with all payment methods, but payments using woocommerce are 100% compatible.

== Frequently Asked Questions ==

= What is the plugin license? =

* This plugin is released under a GPL license.

= How to install the plugin? =

* Download the file in wordpress, or directly in your store at PLUGINS> ADD NEW.
* Right after that, search for the Joy Checkout Plugin, notice the installation and your checkout is already working.

= Where are the settings? =

* The settings you can find in WOOCOMMERCE > CHECKOUT CONFIGURATIONS/CONFIGURAÇÕES DO CHECKOUT

= Can I disable email capture before checkout? =

* Yes, go to WOOCOMMERCE> SETTINGS> ACCOUNTS AND PRIVACIES and then immediately enable the option "Allow your customers to place orders without an account"
